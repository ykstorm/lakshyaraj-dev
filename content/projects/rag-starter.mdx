# RAG Starter

A production-ready starter template for RAG applications using pgvector.

## Performance Targets

- **Latency**: Sub-50ms queries
- **Timeout**: 600ms per request
- **Similarity Threshold**: 0.30 cosine floor

## Technology Stack

- pgvector for vector storage
- Sub-50ms response times
- Configurable similarity thresholds

## Use Case

Ideal for building intelligent search and retrieval systems with PostgreSQL and vector embeddings.
