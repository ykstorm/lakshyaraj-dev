# Tripwire

A mid-stream guardrail for LLM output. A pattern engine watches the token stream
as it generates and aborts the moment a rule trips, with sub-millisecond
decision latency — so unsafe completions are killed before they finish
rendering.

- npm: `@ykstormsorg/tripwire` v1.0.1
- Code: https://github.com/ykstorm/tripwire
