# Quickdraw

A CLI for benchmarking LLM streaming performance. It runs prompt sets against
any model, captures time-to-first-token and tokens-per-second, computes cost per
1K tokens, and diffs every run against a saved baseline.

- npm: `@ykstormsorg/quickdraw` v1.0.1
- Code: https://github.com/ykstorm/quickdraw
