# Stackup

A local Kubernetes lab. `make up` bootstraps a kind cluster preloaded with
ArgoCD, Argo Rollouts, and the Grafana/Prometheus stack — a way to learn real
production k8s and GitOps without a cloud bill.

- Code: https://github.com/ykstorm/stackup
