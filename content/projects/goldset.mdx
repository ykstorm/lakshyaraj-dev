# Goldset

An LLM evaluation harness that runs as a GitHub Action. Three runners — golden
datasets, LLM-as-judge, and structural assertions — score every pull request,
post a delta comment, and block merge on regression.

- npm: `@ykstormsorg/goldset` v0.2.0
- Action: `uses: ykstorm/goldset@v0.2.0`
- Code: https://github.com/ykstorm/goldset
- Marketplace: https://github.com/marketplace/actions/goldset
